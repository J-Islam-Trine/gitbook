# টেবিল অপারেশনস

### টেবিল তৈরি করা

```sql
CREATE TABLE table_name (
COLUMN_NAME COLUMN_TYPE,
COLUMN_NAME COLUMN_TYPE,
COLUMN_NAME COLUMN_TYPE,
.....
....
....
);
```

### টেবিল মুছে ফেলা

```sql
DROP TABLE table_name;
```

### টেবিল দেখা

```sql
SHOW TABLES;
```

### টেবিলে কী কী কলাম আছে তা দেখা

```sql
SHOW COLUMNS FROM table_name;
```

অথবা

```sql
DESC table_name;
```
