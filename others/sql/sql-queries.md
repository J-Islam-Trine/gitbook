# SQL queries

### সব ডাটাবেজের তালিকা

```sql
show databases;
```

### ডাটাবেজ তৈরি করা

```sql
CREATE DATABASE newDatabase;
```

### ডাটাবেজ মুছে ফেলা

```sql
DROP DATABASE newDatabase;
```

### কোন ডাটাবেজটি নিয়ে কাজ করা হচ্ছে তা দেখা

```sql
SELECT database();
```

### ডাটাবেজ বেছে নেয়া

```sql
user newDatabase;
```
